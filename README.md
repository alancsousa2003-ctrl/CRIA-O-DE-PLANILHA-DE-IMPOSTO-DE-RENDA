Esta planilha em excel contém uma ferramenta de imposto renda onde o usuário consegue organizar todas as suas declarações evitando a famosa 'malha fina' pela Receita Federal. No arquivo, é possível inserir informações pessoais, rendimentos e movimentações bancárias, bem como todas as entradas que entram na 'mão do usuário'. Além disso, a planilha possui um design moderno, simples e bem intuitivo no qual se assemelha muito a um site contendo botões para navegação entra as abas.
